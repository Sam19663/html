Css Box model
  • Css Borders
  • Css margins
  • Css padding
  • Css Height
  • Css Weight

borders are used to create visible boundaries around HTML elements. You can control the size, style, and color of borders using CSS properties.

margins are used to control the space around an HTML element. Margins define the amount of space between an element's border and adjacent elements or the containing element. You can control the margin space for each side of an element, and you can use various CSS properties to specify margin values.

the padding property is used to control the space between an element's content and its border. It defines the inner spacing of an element, creating a buffer or gap around the content.

the height property is used to set the height of an element. It specifies the height of the content area of an element, excluding padding, border, and margin.

there is no specific "weight" property for defining the thickness or heaviness of text. Instead, the weight of text is typically controlled through the font-weight property.
